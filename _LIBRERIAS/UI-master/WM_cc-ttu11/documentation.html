A) Getting started - top

Unzip & place

  1. Unzip your downloaded thetooltip.zip file.
  2. Place thetooltip.css in the directory of your website or project.

Link thetooltip.css to your HTML document

  1. Copy the following code:

    <link rel="stylesheet" type="text/css" href="thetooltip.css" />
  2. Paste it within the <head></head> tags of your HTML document.

Creating a tooltip

It is really easy to create a tooltip:

  1. Copy the following code:

    <a href="">anchor<span class="tooltip applegreen">The Tooltip works!</span></a>
  2. Paste it somewhere within the <body></body> tags of your HTML document.
  3. Change the applegreen classname to your desired color (choose one from the classname overview).

Creating a tooltip with an image inside

  1. Copy the following code:

    <a href="">anchor<span class="tooltip applegreen"><img src="filename.jpg" alt="" /></span></a>
  2. Paste it somewhere within the <body></body> tags of your HTML document.
  3. Change the source of the image, filename.jpg to your image's path.
  4. Make sure your image is smaller then the default width (300px) or change the default width to match your image's width.
    Alternatively, if you only want to use tooltips with images inside, you can change the default width to auto.

B) Adding The Tooltip to a Wordpress website - top

To add The Tooltip to your Wordpress website, we will be working in your Wordpress theme's directory. By default this is located at wordpress/wp-content/themes/themename/

  1. Place thetooltip.css inside this directory.
  2. Open header.php
  3. Add the following code between the <head></head> tags of this document: 
    <?php wp_enqueue_style('thetooltip',get_bloginfo('stylesheet_directory') . '/thetooltip.css'); ?>
  4. Now we are ready to use The Tooltip.
  5. Login to your Wordpress admin panel.
  6. Open your post or page you want to add a tooltip to.
  7. Make sure the HTML tab (next to the Visual tab) is activated in the text editor.

  8. Copy the following code:

    <a href="">anchor<span class="tooltip applegreen">The Tooltip works!</span></a>
  9. Paste it in the text editor on the place you want a tooltip to appear.

C) Positioning - top

The Tooltip can be positioned by adding one of the following classnames to it:

  • left
  • center
  • right

By default The Tooltip is positioned left. In this example we are adding the center classname:

<a href="">anchor<span class="tooltip center applegreen">This is a tooltip with center positioning</span></a>

D) Changing the default width - top

The Tooltip's default width is 300px. In this example, we are going to change the width from 300px to 500px:

  1. Open thetooltip.css
  2. Locate .tooltip
  3. Change 
    width:300px;
    to 
    width:500px;
  4. Locate .tooltip.center
  5. To ensure our tooltip centers when we would apply the center classname, we need to change margin-left too. Change 
    margin-left:-165px;
    to 
    margin-left:-265px;

    Formula:

    margin-left = - (width / 2) + padding

    The Tooltip'spadding is always 15. So in this example this comes down to:

    -265 = - (500 / 2) + 15
  6. Save thetooltip.css

E) Using The Tooltip as a speech bubble - top

What if you want to use The Tooltip as a speech bubble, so it would just be visible without the need to hover over an anchor element? It's simple. Just add the speechbubble classname to your tooltip. Example:

<span class="tooltip speechbubble applegreen">This speech bubble is visible without a hover</span>

Note that speech bubbles are not restricted to span elements. You could do the same with a div element and other block level elements.
