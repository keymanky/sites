A) HTML Structure - top

<ul class="stn-menu">
	<li class="hasSubNav hasArrow">
		<a href="javascript:;">Item #1</a>
		<span class="arrow"></span>
		
		<ul>					
			<li><a href="javascript:;">Sub Item</a></li>
			<li><a href="javascript:;">Sub Item</a></li>
			<li><a href="javascript:;">Sub Item</a></li>
			<li><a href="javascript:;">Sub Item</a></li>
		</ul>
	</li>
	<li class="hasSubNav hasArrow">
		<a href="javascript:;">Item #2</a>
		<span class="arrow"></span>
		
		<ul>					
			<li><a href="javascript:;">Sub Item</a></li>
			<li><a href="javascript:;">Sub Item</a></li>
			<li><a href="javascript:;">Sub Item</a></li>
			<li><a href="javascript:;">Sub Item</a></li>
		</ul>
	</li>
	<li><a href="javascript:;">Item #3</a></li>
</ul>

To create a sub menu simply nest an unordered list within an LI tag. Also add the class .hasSubNav to that LI tag.

To add an arrow, add the class .hasArrow to the parent LI tag. Aslo add a SPAN tag with the class .arrow right under the anchor tag.

To select one the the predefined colors for the menu, add one of the following classes (red, green, orange, purple, blue) to the main UL tag.
