b) JavaScript - top

This theme imports three Javascript files.

  1. jQuery
  2. jQuery Flot
  3. myAnalyticsLight plugin
  1. jQuery is a Javascript library that greatly reduces the amount of code that you must write.
  2. This is a jQuery based plotting libary. Webpage: http://code.google.com/p/flot/
  3. This is my custom script. There are set the basic options and there is the connection with the php file. This is draw charts.

D) Api Usage - top

At first, you need to set account datas in "myAnalyticsLight/config.php"

	$username = ""; // Your Google Analytics account. Eg.: info@example.com
	$password = ""; // Your Google Analytics password
	$profileId = "";  // Your Google Analytics Profile id. Eg.: ga:12345678

The Profile ID:

"Google provides some pretty terrible support for finding this number - Finally found it though.
1. To get it yourself you will need to be an admin or ask an admin status user to give it to you.
2. It can be found within the "Edit" section of the specific website profile. From there it's near the top left. If you aren't seeing it , you are possibly in the wrong "edit" section, it is in the SECOND summary level just before clicking any deeper takes you to the dashboard.
3. It should be a 9 digit PURE NUMERICAL number, no UA's or dashes involved"
