myAnalyticsLight

Examples


Basic usage:

$('#placeholder').myAnalyticsLight();

Result:

Deffault values:

var defaults = {
	php : 'myAnalyticsLight/report.php',  // default path to report.php
	tooltip : true,  // enable tooltip or not
	dateFormat : 'Y-m-d',  // date format
	data: [{
		metrics : 'ga:pageviews',  // metrics
		label : 'Pageviews',  // label text
		color : "#0000FF"  // line color
	}]
};

Disable tooltips:

$('#placeholder2').myAnalyticsLight({
	tooltip : false
});

Result:

Custom metrics:

$('#placeholder3').myAnalyticsLight({
	data: [{	
		metrics : 'ga:visitors',
		label : 'Visitors',
		color : "#333333"
	},{
		metrics : 'ga:pageviews',
		label : 'Pageviews',
		color : "#FF0000"
	},{
		metrics : 'ga:visits',
		label : 'Visits',
		color : "#FFFF00"
	} ]
});

Tested metrics

But maybe others are work too.
(From: http://code.google.com/intl/hu/apis/analytics/docs/gdata/gdataReferenceDimensionsMetrics.html)
ga:visitors Total number of visitors to your website for the requested time period.
ga:newVisits The number of visitors whose visit to your website was marked as a first-time visit.
ga:percentNewVisits The percentage of visits by people who had never visited your site before.
(ga:newVisits / ga:visits) * 100
ga:visits The total number of visits over the selected dimension. A visit consists of a single-user session.
ga:timeOnSite The total duration of visitor sessions represented in total seconds. The value returned in the XML is a string and various client libraries use different types to represent this value, such as a double, float, long, or string.
ga:avgTimeOnSite The average duration visitor sessions represented in total seconds.
ga:visitLength / ga:visits
ga:organicSearches The number of organic searches that happened within a session. This metric is search engine agnostic.
ga:entrances The number of entrances to your website. The value will always be equal to the number of visits when aggregated over your entire website. Typically used with ga:landingPagePath
ga:entranceRate The percentage of pageviews in which this page was the entrance.
(ga:entrances / ga:pageviews) * 100
ga:bounces The total number of single-page (or event) sessions to your website.
ga:entranceBounceRate The percentage of single-page visits (i.e. visits in which the person left your site from the entrance page).
(ga:bounces / ga:entrances) * 100
ga:visitBounceRate The percentage of single-page visits (i.e., visits in which the person left your site from the first page).
(ga:bounces / ga:visits) * 100
ga:pageviews The total number of pageviews for your website.
ga:pageviewsPerVisit The average number of pages viewed during a visit to your site. Repeated views of a single page are counted.
ga:pageviews / ga:visits
ga:uniquePageviews The number of different (unique) pages within a visit, summed up across all visits
ga:avgTimeOnPage The average amount of time visitors spent viewing this page or a set of pages.
ga:timeOnPage / (ga:pageviews - ga:exits)
ga:exits The number of exits from your website. As with entrances, it will always be equal to the number of visits when aggregated over your entire website.
ga:exitRate The percentage of site exits that occurred out of the total page views.
(ga:exits / ga:pageviews) * 100
ga:pageLoadSample The number of page load time measurements received over the given time range
ga:pageLoadTime The total page load time over the given time range in milliseconds.
ga:avgPageLoadTime The average loading time for this page in seconds.
ga:pageLoadTime / (ga:pageLoadSample * 1000)
