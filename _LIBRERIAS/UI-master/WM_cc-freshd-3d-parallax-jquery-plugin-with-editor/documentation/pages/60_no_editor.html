01 Editing HTML Parameters without using the Editor

It is also possible to edit the Parallax Parameters / Values via the HTML Source, by eiditing the data variables. Here you find the list of the data variables:

  • data-startx : The Layer Item Position Horizontal when Mouse is moved to the left side of the Document.
  • data-endx : The Layer Item Position Horizontal when Mouse is moved to the right side of the Document.
  • data-starty : The Layer Item Position Vertical when Mouse is moved to the top side of the Document.
  • data-endy : The Layer Item Position Vertical when Mouse is moved to the bottom side of the Document.
  • data-startskew : The Horizontal Skew (deformation of Item) depending on Mouse Position (Best Values from -30 to 30).
  • data-endskew : The Horizontal Skew (deformation of Item) depending on Mouse Position (Best Values from -30 to 30).
  • data-zoomxoffset : The Horizontal Offset depending on the Vertical Zoom Values when mouse is moved to the left side.
  • data-zoomxendoffset : The Horizontal Offset depending on the Vertical Zoom Values when mouse is moved to the right side.

Dont forget to add also the IE8 variables, for older browsers:

  • data-startx_ie8 : The Layer Item Position Horizontal when Mouse is moved to the left side of the Document.
  • data-endx_ie8 : The Layer Item Position Horizontal when Mouse is moved to the right side of the Document.
  • data-starty_ie8 : The Layer Item Position Vertical when Mouse is moved to the top side of the Document.
  • data-endy_ie8 : The Layer Item Position Vertical when Mouse is moved to the bottom side of the Document.
  • data-zoomxoffset_ie8 : The Horizontal Offset depending on the Vertical Zoom Values when mouse is moved to the left side.
  • data-zoomxendoffset_ie8 : The Horizontal Offset depending on the Vertical Zoom Values when mouse is moved to the right side.
