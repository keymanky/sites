00 What files do I need to upload to my server?

Please upload the parallax folder to your server. In this folder you will find the following subfolders containing all of the items content:

  • js
  • css
  • assets

You could use your own jQuery but we recommend loading it directly from the Google ressources (see later in this documentation).


01 Implement the jQuery

Add the following lines to your HTML <HEAD>:

<script type="text/javascript" src="http://ajax.googleapis.com/ajax/libs/jquery/1.6.2/jquery.min.js"></script>


02 Add freshline js and freshline css files to your HTML page

Also in the <HEAD> section:

<script type="text/javascript" src="parallax/js/freshline/jquery.freshline.Parallax.js"></script>
<script type="text/javascript" src="parallax/js/jquery.zclip.min.js"></script>
<link rel="stylesheet" type="text/css" href="parallax/css/settings.css" media="screen" />

If you do not want to use the editor just use the minified version of the freshline parallax script like this:

<script type="text/javascript" src="parallax/js/freshline/jquery.freshline.Parallax.min.js"></script>


03 Create a surrounding DIV for your Parallax World

Put it in the HTML:

<div id="world" class="parallax_world" data-zoommin="100" data-zoommax="120">...</div>

The DIV must have set width and height (any) i.e.:

#world {margin-left:auto; margin-right:auto; margin-top:-50px; width:760px; height:486px; position:relative; z-index:100; }


04 Build the World via the Editor or manually and put in the resulting code

First slice your picture the way that you get all the seperate parts of the moving 3D world. Then put in the picture sources in the HTML in seperate DIVs like this one for a normal element:

<!-- TEMPLATE LAYER ITEM -->
<div id="WRITE-YOUR-ID-HERE" class="parallax_world_class YOUR-STARTUP-ANIMATION" data-startx="0" data-endx="0" data-startx_ie8="0" data-endx_ie8="0" data-starty="0" data-endy="0" data-starty_ie8="0" data-endy_ie8="0" data-startskew="0" data-endskew="0" data-zoomxoffset="0" data-zoomxoffset_ie8="0" data-zoomxendoffset="0" data-zoomxendoffset_ie8="0">
<img src="YOUR-IMAGE-SOURCE">
</div>

Never mind the values other than the img src, the ID of the DIV (this is the name that appears in the Editor) and the Startanimation. You can change everything else later on with the WYSIWYG editor.

Or put in a button element like this:

<!-- TEMPLATE BUTTON LAYER ITEM -->
<div id="WRITE-YOUR-ID-HERE" class="parallax_world_class YOUR-STARTUP-ANIMATION" data-startx="0" data-endx="0" data-startx_ie8="0" data-endx_ie8="0" data-starty="0" data-endy="0" data-starty_ie8="0" data-endy_ie8="0" data-startskew="0" data-endskew="0" data-zoomxoffset="0" data-zoomxoffset_ie8="0" data-zoomxendoffset="0" data-zoomxendoffset_ie8="0">
<div style="opacity: 1; display: block;" id="trash_button" class="p_button">
<a href="http://www.google.de"><a href=""><img src="YOUR-IMAGE-SOURCE"></a></div>
</div>


05 Call the jQuery Plugin to build the World online!

Put this code underneath the 3D world DIV:

<script type="text/javascript">

   $(window).load(function() {
      $.noConflict();
       jQuery('#world').fhparallax({
         editor:'on',
         automatic:0,
         mobile_auto_start:"yes",
         path:'parallax/js/ZeroClipboard.swf'
      })
   });
</script>
