The freshD plugin uses javascript/jQuery to transform a block of HTML markup into a moving 3D World, all customization is done trough the use of HTML5 data attributes and some little CSS. The markup is valid for HTML5 rules and the content will be fully indexed by search engines. For the IE8 you must use special data attributes since Microsoft did not work with standards in this one. The general structure of markup follows this schema:


01 Include DIV container for the 3D World (see next Tab for details)

<div id="world" class="parallax_world" data-zoommin="100" data-zoommax="120">...</div>


Add the minimum and the maximum Zoom of the whole scene here. Meaning how big or small can the world get when zooming in relationship to the images(objects) original size.

 

02 Place your objects in this DIV container

The objects (parts) of your 3D World are placed inside the container you just created. An element looks like this:

<!-- TEMPLATE BUTTON LAYER ITEM -->
<div id="WRITE-YOUR-ID-HERE" class="parallax_world_class YOUR-STARTUP-ANIMATION" data-startx="0" data-endx="0" data-startx_ie8="0" data-endx_ie8="0" data-starty="0" data-endy="0" data-starty_ie8="0" data-endy_ie8="0" data-startskew="0" data-endskew="0" data-zoomxoffset="0" data-zoomxoffset_ie8="0" data-zoomxendoffset="0" data-zoomxendoffset_ie8="0">
<div style="opacity: 1; display: block;" id="trash_button" class="p_button">
<a href="http://www.google.de"><a href=""><img src="YOUR-IMAGE-SOURCE"></a></div>
</div>
