This is how you bind the Plugin to one DIV. Please use the (window).load:

<!--
##############################
- ACTIVATE THE BANNER HERE -
##############################
-->
<script type="text/javascript">

$(window).load(function() {
$.noConflict();
jQuery('#world').fhparallax({
editor:'off',
path:'parallax/js/ZeroClipboard.swf',
automatic:1000,
mobile_auto_start:"yes"
})
});
</script>


You can set the following starting parameter:

  • editor (on,off) - If on starts the editor.
  • path (path to Clipboard helper) - The editor needs a helper flash file. This here is the path to this file. You do not need to change this if you use the original installation folders.
  • automatic - Set the time untill the World starts to move automatically if no mouseover has happened in milliseconds. 0 means no automatic movement.
  • mobile_auto_start (yes/no) - Since there is no mouseover with the Touchdevices you can set this parameter to yes to give the mobile devices the freshD feeling. It starts the automatic movement immidiately and has nothing to do with the other "automatic" parameter.
