A) About - top

Kooltip is a pure CSS tooltip.

Features

  • Pure CSS, no images, no js
  • 2 vertical & 9 horizontal positions
  • 12 color styles (icyblue, skyblue, purple, magenta, red, orange, yellow, olivegreen, grassgreen, brown, grey and black)
  • 10 predefinied width styles (50px - 500px) & auto width (default)
  • Tooltip title
  • 5 CSS animation styles (fade, slide-left, slide-right, slide-up, slide-down) for supported browsers (Safari 4+, Chrome 4+, Opera 10.5+, Firefox 4+)

Browser compatibility

  • Internet Explorer 8+ (Internet Explorer 8 doesn't support rounded borders and shadows. Internet Explorer 9 supports them (except text-shadow), but rounded borders and gradients don't like each other (buggy), that's why ie version doesn't contain gradients. (IE10+ supports css gradients too.).
  • Chrome 4+
  • Firefox 3.5+ (FF3.5 supports shadows, FF3.6+ supports gradients too).
  • Opera 10+ (Opera 11.1+ supports css gradients).
  • Safari 4+

Required files

  • kooltip.css - css file

A) HTML Structure - top

You need to place the following code to the BODY section of the page:.

<a href="#" class="kooltip"><span class="tooltip icyblue bottom center w200 slide-up">Lorem ipsum</span>icyblue tooltip</a>

The link with "kooltip" classname is the parent element. Parent element can be link, span, div, li, etc. This contains the span tag with "tooltip" classname. Inside span tag comes the desired tooltip text.

You can add title for tooltip, adding a span tag with "title" classname inside "tooltip" span.

<a href="#" class="kooltip"><span class="tooltip magenta bottom left-center w150 slide-down"><span class="title">KOOLTIP</span>Donec eu libero sit amet quam egestas semper Donec eu libero sit amet quam egestas semper.</span>magenta tooltip</a>

Using tricks you can use kooltip for input elements, buttons, images (use a container/parent element e.g. div or span)

<span class="kooltip"><span class="tooltip orange top left slide-down">Enter your name (required)</span><input name="kooltip1" type="text" value="" /></span>

C) CSS Files and Structure - top

Kooltip uses the kooltip.css file.
To call the CSS file first you need to place the following lines to the HEAD section of the page: (use the correct path to the file!)

<link href="assets/css/kooltip.css" rel="stylesheet" type="text/css" />

Setting color

You need to place the color classname next to the "tooltip" classname. Possible classnames: "icyblue", "skyblue", "yellow", "orange", "red", "olivegreen", "grassgreen", "brown", "black", "grey", "purple", "magenta".

For example:

<a href="#" class="kooltip"><span class="tooltip icyblue bottom center w200 slide-up">....</a>

Setting width

Kooltip has auto width by default, but you can set predefinied width classes. There are 10 predefinied styles: "w50" (has the inner width of 50px), "w100" (has the inner width of 100px), "w150", "w200", "w250", "w300", "w350", "w400", "w450", "w500".

Tooltip with auto width: (default).

<a href="#" class="kooltip"><span class="tooltip icyblue bottom center slide-up">....

Tooltip with 300px width: (add "w300" classname to span tag).

<a href="#" class="kooltip"><span class="tooltip icyblue bottom center w300 slide-up">....

Horizontal positioning

There are 2 classname for horizontal positioning: "top" and "bottom" (add "top" or "bottom" classname to span tag).

<a href="#" class="kooltip"><span class="tooltip icyblue bottom center w300 slide-up">....
<a href="#" class="kooltip"><span class="tooltip icyblue top center w300 slide-up">....

Vertical positioning

There are 9 classname for vertical positioning: "center", "center-left", "center-right", "left","left-center", "left-right", "right", "right-center", "right-left".

For example: center position

<a href="#" class="kooltip"><span class="tooltip icyblue bottom center w300 slide-up">...

Important note: If you set "center", "left-center", "right-center" classnames for vertical positioning, you must set width classname for correct positioning. These position doesn't work with auto width!!!

CORRECT:

<a href="#" class="kooltip"><span class="tooltip icyblue bottom center w300 slide-up">....

<a href="#" class="kooltip"><span class="tooltip icyblue bottom left slide-up">....
INCORRECT:
<a href="#" class="kooltip"><span class="tooltip icyblue bottom center slide-up">....

CSS animations:

You can set 5 classnames for animate tooltips: "fade", "slide-up", "slide-down", "slide-left", "slide-right".

For example: slide-up

<a href="#" class="kooltip"><span class="tooltip icyblue bottom center slide-up">....
For example: fade

<a href="#" class="kooltip"><span class="tooltip icyblue bottom center fade">....
