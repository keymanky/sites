A) Installation - top

To install Country Lock, you must first upload the contents of the site folder, so that country-lock resides in the home folder of your site ( As seen in the below picture ). Make sure the files are chmodded to 644. Try and access the country-lock folder by accessing http://www.yourdomain.com/country-lock. If you can't access it, make sure you've uploaded it to the home folder of your site.

Next you may rename the country-lock folder you just uploaded to whatever you wish. I am not going to rename it. Go to http://www.yourdomain.com/thefoldernameyouchose/ and put the name of the folder you chose to rename country-lock to in the box. If you did not rename it, simply put "country-lock".

Next enter in the password you wish to use and press Install and your done! If you encounter any issues with installation not addressed in this section, you can try and contact me.

B) Usage - top

Go to http://www.yourdomain.com/country-lock/ and select which countries you wish to manage. In this example, I will be selecting the United States and Russia.

Next, select whether you wish to blacklist the selected countries or whitelist them. As the note says: Blacklisting blocks the specified countries from a page we will see later, and whitelisting only allows the specified countries, once again, into the page we will see later.
