Bases de datos diversas

Descarga
Descripción:

Como parte de nuestra formación en sistemas, necesitamos bases de datos de prueba lo mas reales posibles; por ello realice un proceso inverso, que consiste en transformar los datos de documentos de excel y pdf; para finalmente obtener de nueva cuenta el archivo .sql. Todos los datos de estas bases de datos fuerón obtenidos de la WEB publicados por los propios organismos e instituciones responsables; lo que implica que no existe plagio de información.

Retos:
  • Recopilar la información adecuada.
  • Abstración de la estructura de la base de datos.
  • Eliminación de los datos basura.
  • Transformar los datos en el formato correcto.
  • Elaboración de querys para volcar la información así como actualizarla
Requisitos:

El archivo es un respaldo realizado por el Dump de MySql 10.13 y en algunos casos realizado por PhpMyAdmin por lo que recomiendo instalar este ultimo antes de restaurar las bases de datos.
