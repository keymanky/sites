Interfaz tipo Office 2010

Descarga
Descripción:

Programa con una interfaz muy atractiva tipo Office desarrollado en visual studio 2012, que hace altas, bajas, cambios y busqueda rápida sobre la base de datos de correos de México.

Retos:
  • Crear la base de datos de correos adecuadamente.
  • Creación de la interfaz atractiva.
  • Crear un ejecutable para las ultimas versiones de Windows.
Requisitos:

Platarforma windows compatible con Net Framework 3.5 o superior, Mysql instalado en la ruta C:, restaurar la base de datos de correos y la creación de una tabla del control del sistema. Las instrucciones vienen el el pdf.
